# SimpleNetty

## Add as dependency:
Repository: 
```
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
```
Dependency:
````
        <dependency>
            <groupId>com.github.vantrex</groupId>
            <artifactId>SimpleNetty</artifactId>
            <version>1.0.2-RELEASE</version>
            <scope>compile</scope>
        </dependency>
````
